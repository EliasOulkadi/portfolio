Live API

This site runs on a real Node.js + Express API with JWT authentication and a PostgreSQL database on Supabase. Hit Run — these are live requests, not mocked responses.

GET /api/projects
Returns all projects stored in the database. Public endpoint — no auth required.
// click Run to execute
POST /api/contact
Submits a contact message. Validates input, saves to Supabase and returns the created record ID.
Request body 
{
  "name": "Jane Recruiter",
  "email": "jane@company.com",
  "message": "We'd like to talk about a position."
}
// click Run to execute
POST /api/auth/login
Authenticates a user. Returns a signed JWT token used to access protected endpoints.
Request body (demo account) 
{
  "email": "demo@oulkadi.dev",
  "password": "demo1234"
}
// click Run to execute
GET /api/tasks JWT required
Returns tasks for the authenticated user. Requires a valid JWT in the Authorization header. Run Login first.
// login first to get a JWT, then run this

There's a full admin panel behind authentication — CRUD for tasks, projects and messages.
